While

É a repetição de um bloco de comandos enquanto a condição imposta é verdadeira, caso seja falsa, ele não será executado

While (condição)
{
[bloco de instruções]
}

For

Ele repete um bloco de instruções por um determinado número de vezes pré-determinado, nela, o comando de repetição se resume em: Inicialização, incremento e condição.

Ao contrário do while, o for deve ser utilizado quando souber a quantidade exata de repetições necessárias.

For (inicializações; condição; incrementos)
{
[bloco de instruções]
}
