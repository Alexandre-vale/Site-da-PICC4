Tipos de dados e elementos básicos

Caractere char %c ocupa 1 byte
Inteiro int &d ocupa 4 bytes
Real Float %f ocupa 4 bytes

Scanf

Scanf (“%f”, &x);
O scanf está definido na biblioteca stdio.h e serve como leitor de dados padrão. Para utiliza-lo em um programa devemos usá-lo da seguinte forma: scanf (expressão de controle, lista de variáveis);

Printf

O printf é utilizado para exibição de dados, devemos utilizar ele quando queremos exibir na tela uma informação, a sua sintaxe é a seguinte:

Printf(expressão de saída [, parâmetros adicionais]);

Comando If-Else:

O If-Else é uma seleção de uma ou mais instruções que devem ser executadas somente se determinada condição for satisfeita, ou seja, um comando condicional.

Para utilizar o comando, temos a seguinte sintaxe:
If else If (condição)
{
[Bloco de instrução 1]
}
Else
{
{Bloco de instrução 2]
}

Switch

Permite a tomada de decisão em que a condição tenha um número levado de possibilidades, condições apenas para tipos char e int. Ele reduz a complexidade do if-else
Stich (expression)
{
Case constante1: instrucoes1; break;
Case constante2: instrucoes2; break;
…
Case constanteN: instrucoesN;break;
[default: instrucoes;]
}

Array

Tipo de dado identificador [quantidade máxima de elementos];

Arrays, também conhecido como vetores, é uma estrutura de dados linear, onde se armazena valores, todos do mesmo tipo (char, int, float ou outras)
vetores quando criados, possuem lixos de memória, ou seja, o vetor é preenchido automaticamente pelo compilador
caso o programa não completa todos os vetores, os restantes que não foram completados, receberam o valor zero

Matriz

Matrizes são arrays bidimensionais, onde ela é uma estrutura de dados que armazena um conjunto de valores, todos do mesmo tipo, que podem ser acessados individualmente, devem receber todos os valores com o mesmo tipo de variável.

A declaração dela vai ser: tipo de dado identificador [tamanho1] [tamanho2]
